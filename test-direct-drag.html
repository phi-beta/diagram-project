üîç What to Test

This test verifies that nodes now follow the mouse cursor exactly where you click, with no lag or drift at any zoom level.

Expected Behavior: The node should stay "glued" to the cursor at the exact point where you clicked, regardless of zoom level.

üìã Test Steps

  1. Create a Node: Double-click on the canvas to create a node
  2. Test Default Zoom: Click and drag different parts of the node:
    • Click the center of the node and drag - cursor should stay at center
    • Click the edge of the node and drag - cursor should stay at edge
    • Click the top-left corner and drag - cursor should stay at top-left
  3. Test High Zoom (200-300%): Zoom in and repeat the drag tests
  4. Test Low Zoom (25-50%): Zoom out and repeat the drag tests
  5. Test After Pan: Pan the view and repeat the drag tests

  • The mouse cursor stays exactly where you clicked on the node
  • No lag or drift at any zoom level
  • Behavior is consistent across all zoom levels and pan positions
  • Node appears "stuck" to the cursor at the click point

  • Node lags behind the cursor
